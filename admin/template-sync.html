THEME LAYERS (Color-Changeable)

Prefix: theme-

theme-background theme-accent-bar theme-header-shape theme-footer

How it works:

  1. Name any shape/rectangle layer with theme- prefix
  2. When you generate the frame overlay, these layers are hidden & their positions saved
  3. In the app, users see a "Theme" button to change ALL theme layers at once
  4. All theme layers change to the same color ‚Üí unified look!

‚Üí Users can customize colors WITHOUT API calls (instant!)
