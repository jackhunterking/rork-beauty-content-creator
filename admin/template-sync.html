Naming Conventions Guide

1 Template Name Convention (Recommended)

Recommended format: [Format]-[Type]-[Variant]

Examples:

Portrait-BeforeAfter-Classic Instagram Posts (recommended)
Square-BeforeAfter-Classic Basic square layout
Vertical-BeforeAfter-Labels Stories/Reels
Rules: PascalCase, no spaces, hyphens only. Format: Square/Vertical/Portrait. Type: BeforeAfter/ThreeSlot/FourSlot/Single.

2 Layer Naming in Templated.io (Critical)

Slot Layers (User Photos)

Layer name must contain "slot"

slot-before slot-after slot-1 slot-hero

Watermark Layer (Branding)

Layer name must contain "watermark" - Hidden for premium users

watermark watermark-text watermark-logo

Other Layers (Decorations)

Use descriptive snake_case names

background before_label after_label divider_line

Customizable Background Layers

Layers users can customize with solid colors in the editor

background_bar left_circle accent_shape

Configure in "Background Layer Settings" section below

Quick Checklist

  • ✓ Template ID is correct from Templated.io
  • ✓ All photo slots have "slot" in the layer name
  • ✓ Watermark layer contains "watermark" in the name
  • ✓ Canvas size: 1080x1350 (recommended), 1080x1080, or 1080x1920

Found in the Templated.io URL: app.templated.io/editor/[ID]

Recommended format: [Format]-[Type]-[Variant] (e.g., Square-BeforeAfter-Classic)
